🏨 Hotel Management System

A Java-based Hotel Management System with a JavaFX GUI, built using Object-Oriented Programming (OOP) principles.

---

📌 Features
- Room booking system
- Guest management
- Check-in / Check-out operations
- Room types management (Single, Double, Suite)
- Restaurant service
- File handling (Read / Write)
- GUI interface using JavaFX

---

🛠️ Technologies Used
- Java
- JavaFX
- OOP
